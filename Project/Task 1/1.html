Projects

Railway Booking System

A Java and MySQL railway booking system is a computer program that enables customers to purchase train tickets. The front-end user interface is built using Java, while the database used to hold customer and reservation data is MySQL. Users may look for available trains, pick seats, and book tickets. For effective handling of train reservations, the system takes care of ticket price, seat availability, and creates booking records in the MySQL database.

Crypto World

Crypto World is a React.js-based web application using Firebase authentication. Users may log in using either classic email/password authentication or the ease of Google login. After logging in, users may explore numerous cryptocurrency-related functions, including as monitoring real-time pricing, maintaining their portfolio, and obtaining pertinent news and information. This mix of technology provides bitcoin aficionados with a safe and user-friendly experience.
